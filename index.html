Cross-platform medicine ordering and pharmacy app

A Flutter app for searching medicines, placing preorders, scheduling daily medication reminders, and finding nearby pharmacies. The interesting part is the auth design: the anon key ships inside the compiled web bundle, so the client can't be trusted with table access. Registration and login go through SECURITY DEFINER Postgres functions instead, with Row Level Security on and all direct grants revoked.
